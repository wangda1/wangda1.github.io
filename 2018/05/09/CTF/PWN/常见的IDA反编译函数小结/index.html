  • 常见的避免格式化字符串漏洞函数（对字符串中的 % 进行筛选）

{

const char \result; // rax@6

int i; // [sp+18h] [bp-18h]@1

int v3; // [sp+1Ch] [bp-14h]@1

  • 关于IDA中指针的说明

以上例为例：这段代码反映了一个结构体：


struct ptr {

char des[0x19];

char name[0x1b];

char * ptr;

}


常常在这些反编译的代码中碰到内存空间计算的问题；

  • 对于带有 prefix 的 ptr，计算时按照 prefix的单位进行；
  • 对于不带有 prefix 的 ptr，计算时按照 Byte的单位进行。
  • 计算时以上下起止空间为基准，而不是 可输入的 size 大小
