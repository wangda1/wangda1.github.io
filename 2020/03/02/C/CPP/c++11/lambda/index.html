C++ 11 之 Lambda 表达式

发表于 2020-03-02 | 分类于 C ， CPP ， c++11

lambda 表达式

lambda 表达式是 c++ 11 引入的新特性，c++中 lambda 表达式的声明：

[capture list] (params list) mutable exception-> return type { function body }

  • capture list 为外部捕获参数列表
  • params list 为形参列表
  • mutable 用来修饰是否修改捕获的外部变量
  • exception 异常设定

4. 混合方式
